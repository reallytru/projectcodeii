HTML5 & CSS

Chapter 1: Pages 5 - 8

Key Terms and Concepts:

The "section" tag defines large parts of a webpage. They are containers like a div. A section tag, when used, is supposed to provide on a grand scale an idea of what the content on the page is. Div containers have no implicit meaning alone, sections, however, can give parts of a webpage meaning.

The "article" tag specifies an independent, self-contained content. You may use an article tag to contain a post of some kind, a comment, or a news story on a webpage.

Remember:

HTML elements can have attributes, providing additional information about an element. Attributes usually come in name/value pairs like, name="value". Href in an "a" tag and src in an "img" tag are examples of attributes.

Id and Class names for HTML elements enable specific CSS styling and may give meaning to the content contained.

Furthermore, file structure and organization is key to coding well. Be sure to have your external CSS stylesheet in a CSS folder, any images in an image folder, and html files in your main directory (the main folder that contains all your website files and stylesheets). Eventually, you will create a scripts folder for your jquery and other script files.

Lesson Takeaway:

The key idea of HTML5 is the introduction of semantic elements, meaning elements have meaning rather than being given meaning through a class or id name, as was the case with previous versions of HTML. The purpose of these new elements are to define content and present identifiable data easily for developers, browsers, and search engines.
